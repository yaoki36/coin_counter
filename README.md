# コインカウンター
 カメラに映した硬貨の個数•種類を判別し合計金額を表示する.
# 使い方
 黒い下地の上に硬貨を並べ,それをカメラに写す
# コインのカウント
 硬貨の大きさを大中小に区分けして,穴の有無,色から硬貨の種類を判別してカウントする.
